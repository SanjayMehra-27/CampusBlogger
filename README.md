# Campus-Blogger

## INTRODUCTION :
Campus-Blogger is a web application projects which enables blogging for campus/college students.Using Campus-Blogger web application students can easily share blogs post.

## PURPOSE :
The purpose of this web application project is to enhance student's confidence and growing their skills for writting blogs.This problem usually occurred when you are studying in tier-3 and tier-4 college.It is also occurred when student have lack of writting skill,that's why they couldn't even start blogs.For this purpose i developed this project.

## ABOUT : 
This web application project contains a landing page, login and sign-up page, user profile page, viewsUser's profile page and some other features.
### Features :

This page contains Student's Sign Up Form such as First Name, Last Name, College Id, Mobile Number, Email Id, Password and Branch/Course Name(B.Tech, B.C.A, B.B.A).After filling this form it redirects user to Home/Dashboard Page.

This page contains ragistered student's College Id and Password field.After filling this form uers will be redirect on Dashboard/Home Page.
